IN PROGRESS
--------------------------------------------------------------------------------
v0.7.0 — Subtitle Cleaning Options

  Phase 1 (core logic) — new file: core/cleaner_options.py
    - apply_cleaning_options(subtitle, options) — operates on block content,
      not block existence (different from ad detection engine)
    - strip_music_cues(block) — lines entirely of ♪/♫ or music symbols
    - strip_sdh_annotations(block) — [ALL CAPS IN BRACKETS] / (All Caps Parens)
      convention that distinguishes sound FX from normal dialogue
    - strip_speaker_labels(block) — WORD: or - WORD: at line start (all caps)
    - strip_formatting_tags(block, preserve_italic, preserve_bold)
    - strip_between_brackets(block, types) — individual toggles for {} [] () * #
    - normalize_case(block) — sentence case per line
    - merge_duplicate_cues(subtitle) — collapse consecutive identical blocks
      after other cleaning is applied

  Phase 2 (UI) — changes to gui/app.py and gui/batch_panel.py
    - Collapsible "Cleaning Options" panel in Single File tab below sensitivity slider
    - Same panel in Batch tab
    - Checkboxes with sub-options indented under parent (matching reference screenshot)
    - SDH removal gets visible accessibility warning label
    - Options apply at clean time, not scan time
    - Preview mode shows selected block after cleaning options applied

  Notes:
    - These are content modification features, NOT block removal features
    - SDH warning: "Removing SDH content reduces accessibility for deaf and
      hard of hearing viewers"

DECISIONS & STANDING RULES
--------------------------------------------------------------------------------
  - Check for Updates stays in status bar. Opt-in and visible. Non-negotiable.
  - Settings is a dialog, NOT a tab alongside real features.
  - No emojis on buttons except the gear icon on Settings.
  - Sensitivity slider recommendations: level 4 for TV, level 5 for movies.
  - Version format: v0.x.0 through 1.0.0, then semantic (1.0.1, 1.1.0 etc.)
  - Remux supports MKV (mkvmerge) and MP4/M4V (ffmpeg). No mkvmerge needed for MP4.
  - Extract & Save filename: [video stem].[lang].[ext] e.g. Movie.eng.srt
    SDH tracks: Movie.eng.sdh.srt
  - File queue in Single File shows color only, no stale count badges.
  - SubScrubber is an accessibility tool — SDH removal features must carry
    a visible warning. Never make destructive accessibility options a default.
